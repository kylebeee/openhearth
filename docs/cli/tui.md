# `openhearth tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
openhearth tui
openhearth tui --url ws://127.0.0.1:18789 --token <token>
openhearth tui --session main --deliver
```
