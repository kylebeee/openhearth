# `openhearth pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
openhearth pairing list whatsapp
openhearth pairing approve whatsapp <code> --notify
```
