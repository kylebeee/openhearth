# `openhearth setup`

Initialize `~/.kylebeee/openhearth.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
openhearth setup
openhearth setup --workspace ~/.openhearth/workspace
```

To run the wizard via setup:

```bash
openhearth setup --wizard
```
